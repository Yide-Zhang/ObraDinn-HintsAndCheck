# 《奥伯拉丁的回归》身份与下落提示与查验程序 (ObraDinn Hints & Check)

这是一个帮助玩家游玩《奥伯拉丁的回归》的工具。它支持玩家获取提示并查证自己的答案。支持Windows和MacOS。

## 下载

在**右侧的Release侧栏**中下载**匹配你的操作系统**的压缩包，解压缩即得程序本体。

## 运行

双击运行即可。**双击后需要一段时间（不到半分钟）解压缩资源，请耐心等待**。

## 使用方法

### 获取提示

点击“获取新提示”即可。在直接给出答案的提示的前一条会有提示“下一条提示将告诉你答案”。

### 查验你的答案
- 可以在两侧的框内输入你的答案，并点击“查验”以检验你的答案是否正确。**正确的答案会像游戏中一样被锁定**。
- 你的输入答案会被保存（即使是退出程序之后再次打开）。
- 一个人的下落答案会有多个，提示只会给出一个答案，但进行查验时**会匹配游戏中所有正确的答案**。
- 当一侧的提示展示完毕后，程序会自动锁定**默认**的正确答案。

### 格式化
格式化会将程序复原到初始状态。

## 注意

对于**Windows用户**，程序的状态会保存在相同位置下的`hints_used.json`文件中。

**若你需要保存你的数据，请务必不要删除此文件。**
